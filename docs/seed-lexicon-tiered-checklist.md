# 种子词库分层清单

这份清单是给维护词库时快速参考用的。

它不是完整代码说明，而是把当前 `data/lexicon.seed.json` 里的规则，按风险层级重新翻译成更容易看的产品视角清单。

你可以把它理解成：

- 哪些词一出现就应该更严处理
- 哪些词更适合先打到人工复核
- 哪些词本身偏安全，只是用来帮助系统识别科普语境

---

## 先记住 3 层

### `hard_block`

更像“高危红线”。

一般代表：

- 明显导流
- 未成年人高风险组合
- 明显冒用权威 / 特供 / 政府关联

### `manual_review`

更像“高风险提醒，需要人工再看一眼”。

一般代表：

- 夸大宣传
- 医疗功效承诺
- 化妆品虚假宣传
- 迷信伪科学
- 直播夸张促销和互动诱导

### `observe`

更像“偏安全语境识别”。

它不是风险词，而是帮助系统识别：

- 科普
- 健康沟通
- 教育表达

---

## 1. 硬拦截

这层建议当成“强风险”看待。

### 1.1 导流与私域

这一类属于最典型的小红书高风险表达。

当前硬拦截重点包括：

- `微信`
- `vx / v信 / 微X信`
- `加我`
- `私信我`
- `小窗我`
- `二维码`
- `加薇 / 加微 / 加V`
- `评论区留号`
- `私我发你 / 私信发你`
- `关注后私信`
- `加我好友`
- `店铺链接在简介里`
- `v我XX元`

维护建议：

- 只要是“联系方式 + 转化动作”这类表达，基本都应该继续放在更严层。
- 这类词的误报成本通常比漏报成本低。

### 1.2 未成年人边界

这类是红线型规则。

当前硬拦截包括：

- `未成年`
- `18岁以下`
- `初中生`
- `高中生`
- `小学生`

说明：

- 这类词本身未必单独就违规，但在当前系统里被放在高风险层，是为了更保守地拦住后续敏感组合。

### 1.3 权威背书与认证宣称

这里已经做了二次拆分，其中更危险的一组被提升成了硬拦截。

当前硬拦截包括：

- `国家机关推荐`
- `国家机关专供`
- `领导人推荐`
- `特供`

维护建议：

- 这类词和普通夸大宣传不一样，更接近明显违规背书，适合保持在最严层。

---

## 2. 人工复核

这层不是“直接放过”，而是“需要人工确认”。

### 2.1 绝对化与功效承诺

这是当前体量最大的一组。

代表词包括：

- `Top1`
- `销量冠军`
- `全网第一`
- `全球第一`
- `行业第一`
- `唯一`
- `独家`
- `绝无仅有`
- `史无前例`
- `最好 / 最佳 / 最高级`
- `国家级 / 世界级 / 顶级 / 旗舰级`
- `极致 / 万能 / 无敌 / 完美 / 神奇`
- `100%有效`
- `100%正品`
- `质量免检`
- `免抽检`
- `无需检测`
- `绝对安全`

维护建议：

- 这一组很常见，但也最容易误伤正常夸张表达，所以先放人工复核更稳。

### 2.2 医疗健康风险

这组主要处理普通商品、普通内容里出现的医疗功效承诺。

代表词包括：

- `治疗 / 治愈 / 根治 / 康复`
- `处方 / 药方 / 诊断 / 临床有效`
- `防癌 / 抗癌 / 防治高血压`
- `消炎 / 抗炎 / 杀菌 / 抗菌 / 除菌 / 灭菌 / 消毒`
- `解毒 / 排毒 / 清热解毒 / 去除体内毒素`
- `溶脂 / 吸脂 / 燃烧脂肪 / 不反弹`
- `祛疤 / 丰胸 / 脱敏 / 改善过敏`
- `雌激素 / 雄激素 / 荷尔蒙 / 抗生素`
- `治疗糖尿病 / 降血糖 / 活血化瘀 / 调节免疫力 / 改善睡眠`

维护建议：

- 这类表达很危险，但也容易受行业和语境影响，所以目前保持人工复核层更合理。

### 2.3 低俗挑逗与擦边

代表词包括：

- `看完脸红`
- `太刺激`
- `欲罢不能`
- `让人上头`
- `越看越想`
- `深夜慎点`
- `撩`
- `暧昧拉满`
- `荷尔蒙`
- `羞羞`
- `不可描述`

维护建议：

- 这类表达需要结合标题、封面和整体语境一起看，所以先放人工复核。

### 2.4 步骤化内容

代表词包括：

- `教程`
- `步骤`
- `实操`
- `怎么做`
- `手把手`
- `完整流程`
- `亲测过程`

说明：

- 这类词本身不一定违规，但如果叠加敏感话题，会明显升风险。

### 2.5 成人用品软广

代表词包括：

- `私密好物`
- `成人好物`
- `夜间神器`
- `提升亲密体验`

### 2.6 平台导流变体

这组没有直接提到“微信 / 二维码”，但有明显站外平台和竞品导流倾向。

代表词包括：

- `淘口令`
- `T宝`
- `宝贝链接`
- `猫店`
- `某东`
- `TikTok`
- `某音`
- `音儿`
- `KS`
- `某手`
- `某多多`
- `W博`
- `围脖`
- `某博`
- `小破站`
- `BB站`

维护建议：

- 这类词在“仅提到平台”与“明显导流”之间有灰度，所以保留在人工复核层最稳。

### 2.7 权威背书与认证宣称

这一组比“国家机关专供”稍弱，所以保留在人工复核层。

代表词包括：

- `免检产品`
- `中国驰名商标`
- `人民币图样`

### 2.8 化妆品虚假宣传

代表词包括：

- `特效 / 高效 / 全效`
- `速白 / 一洗白 / 一抹白`
- `零刺激 / 100%温和`
- `彻底解决 / 永不复发 / 永不褪色`
- `丰乳`
- `彻底美白`
- `彻底消除斑点`
- `逆龄 / 永葆青春`
- `100%修复 / 彻底愈合`

### 2.9 迷信与伪科学

代表词包括：

- `带来好运气`
- `增强第六感`
- `化解小人`
- `增加事业运`
- `招财进宝`
- `护身符`
- `平衡正负能量`
- `逢凶化吉`
- `旺人旺财`
- `转富招福`

### 2.10 刺激消费与互动诱导

这组重点是“制造紧迫感”和“诱导互动”。

代表词包括：

- `秒杀`
- `抢爆了`
- `再不抢就没了`
- `史上最低价`
- `错过就没机会了`
- `万人疯抢 / 抢疯了`
- `恭喜获奖 / 领取奖品`
- `一键三连`
- `想要扣1`
- `扣“是”`

### 2.11 直播营销禁语

这类词目前仍放在人工复核层，但要特别留意它和导流词叠加时的风险升级。

代表词包括：

- `全网最低价`
- `买到就是赚到`
- `效果立竿见影`
- `私信我拿折扣`
- `私信我拿优惠`
- `秒杀价`

维护建议：

- 单独出现时先人工复核。
- 如果和私信、加V、站外链接这类词一起出现，实际风险会明显上升。

## 3. 观察项

这一层不是风险词，而是帮助系统识别“偏安全语境”。

### 3.1 教育语境

代表词包括：

- `同意`
- `边界感`
- `关系沟通`
- `身体认知`
- `健康科普`
- `医学科普`
- `心理教育`

说明：

- 这类词的作用不是拦截，而是帮助系统理解“这段内容可能是科普、教育、沟通向”。
- 它只能降低误判概率，不能直接覆盖硬风险词。

---

## 4. 维护这份清单时怎么判断该放哪一层

可以用一个很简单的判断法。

### 更适合放 `hard_block`

如果这个词一出现，通常就已经很接近明确违规，比如：

- 明显私域导流
- 明显政府 / 领导人 / 特供背书
- 明显未成年人高风险线索

### 更适合放 `manual_review`

如果这个词很危险，但仍然需要结合语境，比如：

- 绝对化表达
- 医疗功效
- 化妆品夸大宣传
- 直播促销话术
- 迷信和伪科学

### 更适合放 `observe`

如果这个词主要在帮助系统识别偏安全语境，比如：

- 科普
- 健康
- 沟通
- 教育

---

## 5. 当前维护结论

基于 2026-04-22 这轮更新，当前更适合继续保持的原则是：

1. 导流与政府背书继续严。
2. 医疗、化妆品、迷信、直播夸张促销继续放人工复核层。
3. 平台导流别称不要一下子全提到硬拦截，避免误伤普通讨论场景。
4. 教育语境继续只做观察项，不当成“放行白名单”。
